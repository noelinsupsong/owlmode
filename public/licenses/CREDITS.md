# Open Source Credits

All 20 games on this site are **built from scratch** for RetroArcade.
Game rules referenced are public-domain classics; no third-party game source code
was forked. UI assets use only emoji (Unicode standard) or self-drawn primitives.

## Games (built from scratch)

1. Tic-Tac-Toe — classic 3x3 game (Public Domain rules)
2. Rock-Paper-Scissors — Public Domain rules
3. Memory Match — original implementation
4. Snake — inspired by classic arcade Snake
5. Whack-a-Mole — inspired by classic carnival game
6. 2048 — implementation of the classic 4x4 sliding puzzle (Gabriele Cirulli's 2048 is MIT, but this implementation is independent)
7. Minesweeper — classic Microsoft-style cell game, independent implementation
8. Breakout — Atari-style brick breaker, independent implementation
9. Sudoku — classic 9x9 number puzzle, with built-in generator (backtracking solve)
10. Pong — Atari-style paddle game, independent implementation
11. Block Stack — falling-block puzzle (avoiding the trademark "Tetris", original implementation)
12. Flappy — pipe-dodger arcade, independent implementation
13. 15 Puzzle — sliding tile puzzle, Public Domain
14. Connect Four — classic disc-drop game, independent implementation with simple AI
15. Simon Says — color memory pattern game, independent implementation
16. Maze Muncher — maze chase (avoiding the "Pac-Man" trademark), independent implementation
17. Asteroid Shooter — vector-style space shooter, inspired by classic Asteroids
18. Solitaire — simplified Klondike (foundation only), independent implementation
19. Bubble Shooter — color-matching bubble pop, independent implementation
20. Space Invaders — classic alien shooter, independent implementation
